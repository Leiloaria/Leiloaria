# Leiloaria-Back

Projeto para disciplinade Programação Web 2025-2, ministrada pelo Prof. Igor Vanderlei.

Equipe de trabalho: Davy Eugênio, Juan Gustavo e Lucas Emanoel

Trata-se do backend de um sistema de gerenciamento de uma Leiloaria de items em geral. Projeto desenvolvido com Java 25 com Maven e SpringBoot 4. 

Execução direta:
  mvn clean install
  
  mvn spring-boot:run

Execução via Docker:
  docker compose up --build
